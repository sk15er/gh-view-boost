# GitHub Profile View Booster

A fast Python script for learning purposes to send large numbers of requests to a GitHub profile view counter (komarev/gh-pvc).

> **For educational and testing purposes only.**

## Features

- Ultra-fast asynchronous HTTP requests
- High concurrency support
- Two versions: `aiohttp` and `httpx` (HTTP/2)
- Minimal overhead (no unnecessary body reading)
- Clean statistics and performance measurement
